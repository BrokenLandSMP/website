## 🚀 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

访问 `http://localhost:3000` 查看页面。

### 构建生产版本

```bash
pnpm build
```

输出文件位于 `dist/public` 目录，可直接部署到 Vercel。
